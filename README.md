## 기술 스택
**Backend** <br>
- Java 17
- SpringBoot 3.x
- Spring Data JPA (Hibernate)
- MySQL
- Thymeleaf

**Frontend**
- HTML / CSS / JS (Thymeleaf 기반)
- Flatpickr

**Infra**
- AWS EC2 (배포 서버)
- MySQL (AWS 서버 내 설치)
